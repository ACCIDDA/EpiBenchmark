# `epibench plot`

`epibench plot` is a command that, generates visualizations from the score `.csv` file produced by the `epibench score` pipeline. Given a YAML configuration file, the command reads the specified score file and creates plots for the evaluation metrics contained in the `.csv` file.

Users can configure:

- the location of the input score `.csv` file
- the output directory where generated plots will be saved

## Config file 

To run `epibench plot`, you will have to create a YAML configuration file containing two top-level sections: `data` and `plot_paths`. 

### data

The data section specifies the input data files required for plotting.

#### score_file_path

Path to the input .csv score file generated by EpiBench.

Requirements:
- The file must exist.
- The file must be a valid .csv file.

### plot_paths

The `plot_paths` section specifies where generated plots should be saved.

#### plot_output_dir

Path to the output directory for saving plots.

Requirements:

- If the directory does not exist, EpiBench will create it automatically.
- If the path already exists, it must be a directory. 


## Example:
```yaml
data:
  score_file_path: <path_to_csv_file>

plot_paths:
  plot_output_dir: <path_to_save_plots>
```
